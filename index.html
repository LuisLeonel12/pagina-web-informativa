SHARK INVENTORY INTELIGENT

PANTALLA DE CARGA: muestra una pantalla de carga colorida y un diseño distinto con un contador colocado arriba de la barra de carga y una imagen con movimiento que simule la carga del software.

REGISTRO DE USUARIOS: ingrese todos los datos para poder darse de alta en la base de datos y poder guardar su correo, contraseña y poder cifrar la contraseña para que nadie pueda acceder a los datos.

LOGIN: ingrese su correo y contraseña con la que anteriormente se hizo en el registro de usuarios.

MENU PRINCIPAL: muestra una interfaz con las siguientes pestañas productos, clientes, categorías, usuarios, orden.

PRODUCTOS: se agregan los productos con todos los datos correspondientes como el nombre del producto, la cantidad, la marca, precios, característica entre otras cosas.

CLIENTES: en este apartado se guardan los datos de los clientes para poder realizar un seguimiento o incluso para hacer Email Marketing.

CATEGORIAS: en este bloque se muestran todas las categorías de productos que se pueden llegar a manejar, incluyendo las marcas

USUARIOS: en esta sección se muestran los usuarios que están registrados en la base de datos

ORDEN: en esta pestaña se muestra una interfaz en donde se lleva a cabo la creación de la orden de compra con los datos del producto, el precio, la cantidad y la marca y un precio total
